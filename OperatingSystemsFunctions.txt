Security is an important function of computer Operating Systems.  Security is what protects the operating system and the information within the system.  This consists of passwords, protection from unauthorized links, and anything involving personal information or security.  Security is the most important first step to an Operating System and would not work without it.
Memory is another key element of the Operating System and it's function.  There are multiple types of memory systems in the computer.  The main memory is what the CPU uses to function and it is the most important part of a computer.  Other memories devices are stored in the Operating System but are not all controlled by the CPU.
